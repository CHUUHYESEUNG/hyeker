# 11번가 콘텐츠 기획

## 프로젝트 소개

모바일 앱 디자인 프로젝트로, **사용자 중심**의 직관적인 경험을 제공하는 것이 목표였습니다.

![모바일 앱 메인 화면](/mobile_1.png)

> 모바일 환경에 최적화된 직관적인 인터페이스

## 주요 기능

### 📱 모바일 최적화

- 반응형 레이아웃
- 터치 친화적 UI
- 빠른 로딩 속도

### 🎨 비주얼 디자인

현대적이고 미니멀한 디자인 언어를 사용했습니다.

#### 디자인 원칙

1. **Simple**: 불필요한 요소 제거
2. **Clear**: 명확한 정보 전달
3. **Beautiful**: 아름다운 비주얼

## 디자인 시스템

### 스페이싱

| 크기 | 값 |
|------|-----|
| xs | 4px |
| sm | 8px |
| md | 16px |
| lg | 24px |
| xl | 32px |

### 아이콘

Lucide Icons를 기본으로 사용하고, 필요에 따라 커스텀 아이콘을 제작했습니다.

## 사용자 플로우

```
홈 → 탐색 → 상세보기 → 액션 → 완료
```

각 단계에서 사용자가 다음 행동을 쉽게 예측할 수 있도록 설계했습니다.

## 프로토타입

Figma에서 인터랙티브 프로토타입을 제작하여:

- 실제 사용자 테스트 진행
- 피드백 수집 및 반영
- 개발팀과 원활한 협업

## 모션 디자인

After Effects를 활용한 마이크로 애니메이션:

1. **Onboarding** 애니메이션
2. **Loading** 스테이트
3. **Transition** 효과
4. **Success** 피드백

> 모든 애니메이션은 자연스럽고 의미있는 움직임을 추구합니다.
